# 🍕 Pizza El-Mohandes - Premium Restaurant Website

A modern, high-end, dark warm-themed restaurant landing page featuring an interactive custom pizza builder. Built with performance-first vanilla JavaScript and highly optimized CSS graphics.

👉 **[Live Demo](https://khaled24ao.github.io/pizza-elmohandes-app/)** _(قم بتحديث الرابط بعد تفعيل GitHub Pages)_

---

## ✨ Key Features

* **Premium UI/UX:** An appetite-inducing dark warm color palette with exceptional Arabic/Bilingual typography utilizing *Cairo* and *Tajawal* web fonts.
* **Interactive Pizza Builder:** A responsive visual staging area tracking selected sauce bases and scattering toppings dynamically using a custom mathematical circle-distribution algorithm.
* **Micro-interactions:** Hardware-accelerated animations (`toppingPop`, `slowSpin`, `pulse`) and cart bump indicators for fluid feedback.
* **Fully Responsive UI:** Complete mobile navigation drawer adjustments and flexible CSS grids tailored for desktop, tablet, and mobile views.

## 🛠️ Tech Stack

* **Structure:** Semantic HTML5
* **Styling:** CSS3 (Custom Properties, Flexbox, Dynamic Grids, `@keyframes` Animations)
* **Interactions:** Vanilla JavaScript (ES6+ Native DOM manipulation)
